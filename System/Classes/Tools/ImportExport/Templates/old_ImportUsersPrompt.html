This page allows you to import users into the system. The file uploaded should be in tab delimited format. This option is available from Excel's 'Save As...' option 'Save as Type' - 'Text (Tab Delimited)'.

The first row of your file should contain headers for each column. These headers are important as this is how the system will identify which fields the data will be loaded into. The subsequent rows should contain the details of each user. An example is shown below:

First Name Last Name Email
JohnSmithjohn.smith@innovativemedia.co.nz
JaneDoejane.doe@innovativemedia.co.nz

If your spreadsheet contains a single name field instead of separating the persons names into a first and last name, please use the title "Full Name" so the system will recognise the field and separate them into a first and last name. For example:

Full Name Email
John Smithjohn.smith@innovativemedia.co.nz
Jane Doejane.doe@innovativemedia.co.nz
