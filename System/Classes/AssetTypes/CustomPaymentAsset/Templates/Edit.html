Custom Display Template

You may define a custom template for your form instead of simply having the field names in one column and the fields in another column. If you would like to use the template function, please tick this box: {tmpl_fieldset_field fieldset="FieldSet" field="AST_CUSTOMPAYMENT_USE_CUSTOM_DISPLAY_TEMPLATE"}

To create a template, you need to insert special tags where you would like your fields to appear. To add a tag, simply surround the field name with square brackets. e.g. [Name]. Also be sure to add a submit button by entering a [Submit] tag in your template.
